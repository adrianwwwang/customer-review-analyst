How it works

Six steps,
one conversation

The skill orchestrates the AI through a structured workflow — no scripts to run, no dashboards to configure.

01
Gather inputs
URL or product name, time range, desired output formats, and save location.
02
Fetch review data
Paginates 50+ reviews from Trustpilot, Amazon, App Store, or Google Play.
03
Analyze & classify
Sentiment scoring, monthly aggregation, complaint theme extraction with quotes.
04
Generate HTML
Runs generate_html.py to produce a self-contained interactive dashboard.
05
Optional outputs
Generates PowerPoint slides and/or a PDF report if requested.
06
Deliver summary
File paths, sentiment split, top 3 complaints, and top 3 action items — inline.

Up in

Copy the skill to your AI assistant's skill directory, restart, and speak naturally.

Auto-detected on restart
Works across all projects
pip deps installed on demand
Trigger with natural language
bash
# Clone the repository git clone https://github.com/adrianwwwang/customer-review-analyst.git # Install the skill globally cp -r customer-review-analyst/skills/customer-review-analyst ~/.claude/skills/

Restart Claude Code — the skill is auto-detected in all projects.

bash
# Option A — personal skill (all projects) cp -r customer-review-analyst/skills/customer-review-analyst ~/.copilot/skills/ # Option B — project-level skill mkdir -p .github/skills cp -r skills/customer-review-analyst .github/skills/ # Option C — custom instructions (always-on) cp github-copilot/copilot-instructions.md .github/copilot-instructions.md

Works in VS Code, Eclipse, Xcode, and JetBrains.

bash
